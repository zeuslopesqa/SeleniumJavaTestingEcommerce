About:
This project was created to study and improve my skills about Selenium and Java in scenario of an e-commerce website. I've developed automatizated tests to simulate a group of regression tests. My  mainly goal is improve my skills in the tecnologies mentioned.

How to use:
To use this application, it's necessary to have in your computer Chrome Web Driver/chrome downloaded and all the project cloned in a IDE.

Links:
Chrome Web Driver: https://chromedriver.chromium.org/getting-started

Chrome: https://www.google.com/intl/pt-BR/chrome/

Tips:
To test in another navigator you need only change the parameters in the code from chrome to another one navigator(Ex: Firefox).
